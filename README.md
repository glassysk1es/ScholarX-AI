# 🎓 ScholarX: The Decentralized Student OS

ScholarX is an AI-powered personal assistant designed specifically for Indonesian university students. Built on **Solana**, powered by **ElizaOS**, and hosted on **Nosana's** decentralized GPU network.

### 🌟 Features
- **Smart Workflow:** Automated task breakdown from syllabuses.
- **Deep Academic Research:** High-quality reference sourcing via AI.
- **Proof of Productivity:** Earn XP and on-chain badges for every task completed.
- **Low-Cost Intelligence:** Affordable AI access through decentralized infrastructure.

### 🛠 Tech Stack
- **AI Framework:** ElizaOS
- **Inference:** Nosana Network (DeAI)
- **Blockchain:** Solana
- **Language:** TypeScript
